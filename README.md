# 📚 Books Manager - Django CRUD Project

This is a simple Django-based web application that allows users to **Create**, **Read**, **Update**, and **Delete (CRUD)** books.

---

## 🔧 Features

- Add new books with title, author, and description.
- View a list of all books.
- Edit/update book details.
- Delete books with confirmation.
- Clean and minimal UI using Django templates.

---

## 🛠️ Tech Stack

- Python 3
- Django (Web framework)
- HTML/CSS (for templates)
- SQLite (default database)
